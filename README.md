## Quick Start

### 1. Install Claude Nine
```bash
git clone https://github.com/playajames760/claude-nine.git
cd claude-nine
./install.sh
```

### 2. Start Your First Session
```bash
cd your-project

/claude-nine-prompt
```

